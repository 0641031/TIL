### Adjacent sibling combinator (+)

> 인접 형제 선택자(+)는 앞에서 지정한 요소의 바로 다음에 위치하는 요소만 선택합니다.


```css
input[type=checkbox] + label {
  padding-left: 20px;
  margin-right: 20px;
}
```



checkbox와 label이 아니고, checkbox 바로 다음에 위치하는 label**만** 선택한다.
